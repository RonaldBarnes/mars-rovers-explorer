# Mars Rovers

## Usage
<h4>Set up & Start the Rails Server</h4>
1. Fork & clone this repository
2. cd backend
3. bundle install
4. rails db:reset
5. rails db:seed
6. rails s
<h4>View the Front End in Browser</h4>
7. cd ../frontend
8. npm install
9. npm start
10. Run `npm install`
11. Run `npm start`
