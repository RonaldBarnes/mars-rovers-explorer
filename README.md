# Mars Rovers

## Usage
<h4>Set up & Start the Rails Server</h4>
<p>1. Fork & clone this repository</p>
<p>2. cd backend</p>
<p>3. bundle install</p>
<p>4. rails db:reset</p>
<p>5. rails db:seed</p>
<p>6. rails s</p>
<h4>View the Front End in Browser</h4>
<p>7. cd ../frontend</p>
<p>8. npm install</p>
<p>9. npm start</p>
<p>10. Run `npm install`</p>
<p>11. Run `npm start`</p>
